### 技术栈选择 ✅

**决策依据**:
- **L1-STREAMLINED**：快速迭代，~500行/文档文件
- **零成本/开源偏好**：React、Node.js、MongoDB、Redis均为开源技术
- **快速原型验证**：单体架构 + 模块化设计，适合快速迭代

**技术栈**:
- **前端**: React + TypeScript + Ant Design
- **后端**: Node.js + Express
- **数据库**: MongoDB（主存储）+ Redis（缓存）
- **部署**: Docker容器化
- **AI服务**: GLM-4 API
- **语音服务**: Web Speech API

**架构模式**: 单体架构 + 模块化设计
- **理由**: 项目规模适中，初始开发成本最低；符合L1-STREAMLINED快速迭代理念；模块化设计便于后续扩展
