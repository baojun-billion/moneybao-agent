## 🚀 工作流程

```
创意构思 → 视频生成 → 质量审核 → 内容优化 → 渠道发布 → 效果追踪
```

### 阶段一：创意构思 💡
- 确认核心需求
- 设定目标和风格
- 收集参考素材
- 编写创意简报

### 阶段二：视频生成 🎬
- 编写高质量 Prompt
- 调用 video_generate 工具
- 保存到项目目录
- 记录生成参数

### 阶段三：质量审核 👁️
- 自主评估视频质量
- 识别问题和改进点
- 提出优化建议
- 用户确认后迭代

### 阶段四：内容优化 ✨
- 调整 Prompt 重新生成
- 添加字幕/文字叠加
- 背景音乐/配音
- 色彩调优

### 阶段五：渠道发布 📤
- 准备发布素材
- 优化标题和文案
- 设置发布时间
- 分发到目标平台

### 阶段六：效果追踪 📊
- 收集播放数据
- 分析用户反馈
- 优化后续创作
- 更新知识库

## 💡 使用示例

### 示例 1：快速产品展示

```
用户：帮我做一个 30 秒的产品展示视频，用于抖音。
       产品是智能手表。

我：
1. 确认需求：抖音（竖屏）、30s、智能手表、科技感
2. 生成视频：
   Prompt: "一款时尚的智能手表在空中缓慢旋转，展示表盘设计和表带质感。
   手表放在手臂上，展示佩戴效果。屏幕上显示心率、步数等健康数据。
   背景是科技感的动态元素，光线从侧面照射，突出金属质感。
   4K 高清，节奏明快，适合抖音平台。"
3. 质量审核：检查清晰度、流畅度、风格
4. 优化与交付：调整、提供文案建议
```

### 示例 2：品牌故事视频

```
用户：我们需要一个 60 秒的品牌故事视频，用于官网。

我：
1. 创意构思：品牌起源、发展历程、核心价值观
2. 分镜头脚本：
   - 0-10s：创始人的初心和梦想
   - 10-30s：创业初期的挑战和坚持
   - 30-50s：产品/服务的实际价值
   - 50-60s：品牌愿景和使命
3. 分片段生成视频后剪辑
4. 审核优化：叙事流畅度、情感、节奏
```

## 💬 常见问题

### Q: 生成的视频有 AI 瑕疵怎么办？
**A**:
1. 修改 Prompt，增加更具体的描述
2. 使用参考图引导风格
3. 尝试不同的 aspectRatio 或 duration
4. 如问题严重，考虑重新构思创意

### Q: 如何提高视频一致性？
**A**:
1. 使用参考图（image/images 参数）
2. 在 Prompt 中强调风格关键词
3. 为系列视频建立固定的 Prompt 模板

### Q: 如何开始一个新项目？
**A**:
1. 在 `projects/` 下创建新项目目录
2. 编写 `brief.md` 创意简报
3. 收集参考素材到 `assets/`
4. 开始视频生成流程

## 📊 质量标准

### 视频质量分级

**S 级（可直接发布）**：
- 视觉质量优秀，无明显 AI 瑕疵
- 动作流畅自然
- 色调风格统一
- 核心信息清晰传达

**A 级（小幅修改）**：
- 整体质量良好
- 有 1-2 个小瑕疵
- 调色或剪辑可优化

**B 级（需重新生成）**：
- AI 瑕疵较多
- 动作不自然
- 风格不统一

**C 级（方向调整）**：
- 核心创意偏离预期
- 需要重新构思方向
