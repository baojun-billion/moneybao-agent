## 🚨 常见问题

### Q1：点击快捷按钮没有反应

**可能原因**：处理中，按钮被禁用

**解决方法**：等待当前问题处理完成

---

### Q2：AI 回复为空

**可能原因**：API 调用失败或返回异常

**解决方法**：
1. 打开开发者工具（F12）
2. 查看 Console 标签
3. 查看是否有错误信息
4. 复制错误信息给我

---

### Q3：语音不朗读

**可能原因**：浏览器不支持语音合成或权限问题

**解决方法**：
1. 检查浏览器是否支持（推荐 Chrome/Edge）
2. 检查音量设置
3. 查看 Console 标签是否有语音合成错误

---

### Q4：API 调用失败（401 错误）

**可能原因**：API Key 错误或无效

**解决方法**：
1. 确认 API Key 是否正确
2. 登录 GLM 平台，重新生成 API Key
3. 更新 GLM版本.html 中的 API Key

---

### Q5：API 调用失败（429 错误）

**可能原因**：请求频率过高

**解决方法**：
1. 等待几秒后再试
2. 检查账户余额
3. 升级账户套餐

---

### Q6：API 调用失败（500 错误）

**可能原因**：GLM 服务器错误

**解决方法**：
1. 等待几分钟后重试
2. 查看GLM 平台公告
