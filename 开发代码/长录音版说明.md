## 📋 新功能说明

### 1. 录音计时器

**位置**：录音按钮下方

**显示格式**：`00:00`（分:秒）

**更新频率**：每秒更新一次

**示例**：
```
00:05  → 录音 5 秒
00:30  → 录音 30 秒
01:00  → 录音 1 分钟
02:30  → 录音 2 分钟 30 秒
```

---

### 2. 实时识别结果

**位置**：对话上方

**显示内容**：
- 正在识别的内容（临时结果）
- 实时更新

**示例**：
```
正在识别...
你
你好
你好，我想了解
你好，我想了解一下沃橙信息
```

---

### 3. 30秒静音自动停止

**逻辑**：
- 开始录音后，启动静音计时器
- 每次检测到语音，重置计时器
- 如果 30 秒内没有语音，自动停止录音

**提示**：
```
[16:30:00] 30秒无说话，自动停止录音
```

---

### 4. 手动停止录音

**操作**：
- 录音过程中，点击橙色按钮 ⏹
- 录音立即停止
- 处理识别结果

**日志**：
```

[16:30:00] 录音结束
[16:30:00] 开始调用 GLM-4 API...
```

## 🚨 注意事项

### 1. 录音时长限制

虽然可以持续录音 1 分钟以上，但建议：

- **日常使用**：5-30 秒
- **长问题**：30-60 秒
- **特殊情况**：1-2 分钟

### 2. 浏览器限制

部分浏览器可能有以下限制：

- Chrome：无明确限制
- Edge：无明确限制
- Safari：可能有 1 分钟限制

### 3. 系统资源

长时间录音会占用更多系统资源：

- CPU 使用率增加
- 内存使用增加
- 建议适时停止录音
