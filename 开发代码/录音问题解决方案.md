# 🚨 录音问题解决方案

## 问题原因

如果录音功能无法使用，可能是因为：

### 1. 浏览器权限问题

**症状**：点击麦克风按钮没有反应，或提示"无法访问麦克风"

**原因**：浏览器没有麦克风权限

**解决方法**：

**Chrome / Edge**：
1. 点击地址栏左侧的锁图标 🔒
2. 找到"麦克风"设置
3. 选择"允许"
4. 刷新页面

**Firefox**：
1. 点击地址栏左侧的图标
2. 找到"麦克风"权限
3. 选择"允许"
4. 刷新页面

---

### 2. 浏览器不支持语音识别

**症状**：提示"浏览器不支持语音识别"

**原因**：浏览器版本太旧或使用的是不支持的浏览器

**解决方法**：

✅ **推荐使用**：
- Chrome 25+（推荐）
- Edge 79+（推荐）
- Safari 14.5+
- Opera 27+

❌ **不支持或部分支持**：
- Firefox（部分支持）
- Internet Explorer（不支持）

**下载 Chrome**：https://www.google.com/chrome/

**下载 Edge**：https://www.microsoft.com/edge

---

### 3. 麦克风硬件问题

**症状**：可以点击录音，但识别不到文本

**原因**：麦克风未连接、未选择或损坏

**解决方法**：

**检查麦克风连接**：
1. 确保麦克风已连接到电脑
2. 检查麦克风是否正常工作
3. 尝试更换麦克风

**检查系统设置**：
1. 打开"设置" → "系统" → "声音"
2. 检查"输入"设备是否选择了正确的麦克风
3. 调整音量级别

**检查浏览器设置**：
1. 打开浏览器设置
2. 搜索"麦克风"
3. 确认选择了正确的麦克风

---

### 4. 环境安静问题

**症状**：识别不准确，识别出乱七八糟的内容

**原因**：环境噪音太大

**解决方法**：
- 在安静的环境中使用
- 说话时靠近麦克风（10-20cm）
- 说话清晰、慢速、大声
- 使用降噪麦克风
