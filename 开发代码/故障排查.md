# 🚨 遇到问题？手动启动指南

如果 `start.bat` 无法正常运行，请按照以下步骤手动启动：

---

## 🔧 手动启动步骤

### Step 1: 打开命令行

按 `Win + R`，输入 `cmd`，按回车

### Step 2: 进入项目目录

复制以下命令，粘贴到命令行，按回车：

```bash
cd C:\Users\jovi_\.openclaw\workspace-moneybao-agent\开发代码
```

### Step 3: 检查 Node.js

```bash
node --version
```

**如果显示版本号（如 v18.19.0），继续下一步**

**如果提示"不是内部或外部命令"**：
- 访问 https://nodejs.org
- 下载并安装 Node.js（LTS 版本）
- 重启命令行，再次运行 `node --version`

### Step 4: 安装依赖

```bash
npm install
```

**如果报错**，尝试：

```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Step 5: 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:5173

## 🐛 常见问题

### Q: 提示 "不是内部或外部命令"

**A**: Node.js 未安装或未配置环境变量

**解决方法**：
1. 访问 https://nodejs.org
2. 下载并安装 Node.js（LTS 版本）
3. 重启命令行
4. 运行 `node --version` 检查

---

### Q: npm install 报错

**A**: 网络问题或 npm 镜像源问题

**解决方法**：

**方法 1: 切换到国内镜像**
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

**方法 2: 删除 node_modules 重新安装**
```bash
rd /s /q node_modules
npm install
```

**方法 3: 清除 npm 缓存**
```bash
npm cache clean --force
npm install
```

---

### Q: 端口 5173 被占用

**A**: 其他程序占用了该端口

**解决方法**：

**方法 1: 查找并关闭占用端口的程序**
```bash
netstat -ano | findstr :5173
```
记下 PID（最后一列的数字），然后：
```bash
taskkill /F /PID <PID>
```

**方法 2: 修改端口**
编辑 `vite.config.js`，添加：
```javascript
export default defineConfig({
  plugins: [react()],
  server: {
    port: 3000  // 修改为其他端口
  }
});
```

---

### Q: 启动后无法访问 http://localhost:5173

**A**: 防火墙或浏览器问题

**解决方法**：

**方法 1: 检查防火墙**
- 关闭 Windows 防火墙临时测试
- 或允许 Node.js 通过防火墙

**方法 2: 更换浏览器**
- 使用 Chrome 或 Edge 浏览器
- 清除浏览器缓存

**方法 3: 检查命令行输出**
- 查看 Vite 是否正常启动
- 应该显示 "Local: http://localhost:5173"

---

### Q: .env 文件读取不到

**A**: .env 文件位置或格式问题

**检查清单**：

1. `.env` 文件是否在项目根目录（与 `package.json` 同目录）
2. `.env` 文件名是否正确（注意前面有个点）
3. `.env` 文件内容格式是否正确（无引号，无空格）

**正确的 .env 格式**：
```env
VITE_MINIMAX_API_KEY=sk-api-xxx
VITE_ZAI_API_KEY=xxx
VITE_LLM_MODEL=minimax-m2.5
```

**错误的格式**：
```env
VITE_MINIMAX_API_KEY="sk-api-xxx"  # ❌ 不能有引号
VITE_MINIMAX_API_KEY = sk-api-xxx  # ❌ 等号两边不能有空格
```
