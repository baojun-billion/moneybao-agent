## 📋 使用说明
- **推荐工具**：Midjourney V6（质量最高）/ DALL·E 3（最稳定）/ Stable Diffusion XL（最自由）
- **分辨率设置**：Prompt 中已包含 `--ar 1:1`（正方形），无需额外修改
- **风格参数**：Prompt 中已包含 `--style raw`（禁用 Midjourney 默认风格），确保输出符合原始描述
- **负面提示词**：每次生成时，建议同时加入负面提示词，避免低质量输出

### 2. 文件属性
- **格式**：PNG-24（支持透明背景）
- **分辨率**：DPI 72（屏幕显示）或 300（印刷用途）
- **文件大小**：≤ 500KB（头像）/ ≤ 2MB（配图）

### 3. 版本管理
- **v1.0**：初版，用于测试
- **v1.1**：微调（如颜色、细节）
- **v2.0**：重大改版（如风格重设）

## 🔧 高级技巧

### 1. Midjourney V6 参数
```
--ar 1:1        # 宽高比
--style raw     # 禁用默认风格，完全遵循 Prompt
--v 6.0         # 使用 V6 模型（最高质量）
--q 2           # 高质量模式
--s 250         # 风格化程度（0-1000，建议 100-300）
```

### 2. 批量生成
```
如果你需要多张变体，在 Prompt 后面加上 `--seeds [随机数]`，如：
--seeds 12345, 67890, 54321
```

### 3. 局部重绘（Midjourney V6）
```
使用 `/remix` 命令，选中图片后修改 Prompt 中的特定部分，如：
把 "wearing a blue suit" 改成 "wearing a red suit"
```
