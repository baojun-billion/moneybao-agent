## 🚨 常见问题

### Q1：如何生成头像？
**A**：打开 `PROMPTS.md` 第 1 节，复制英文 Prompt 到 Midjourney，生成后用 `QUALITY_CHECKLIST.md` 验收。

### Q2：Logo 需要矢量版本吗？
**A**：必须。Logo 必须同时提供 PNG（栅格）和 SVG（矢量）版本，确保无损缩放。

### Q3：文件大小超标怎么办？
**A**：使用 TinyPNG 在线压缩（无损），或在 Photoshop 中调整导出质量（80%-90%）。

### Q4：色彩偏差如何修正？
**A**：使用吸管工具检查 RGB 值，在 Photoshop 中通过"图像 > 调整 > 色相/饱和度"微调，重新导出时关闭色彩管理。

### Q5：如何检查字体授权？
**A**：使用 Font Squirrel 确认字体许可，将授权文件存入 `assets/fonts/licenses/`。
