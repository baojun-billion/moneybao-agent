## ✅ 核心检查项（Critical Checks - 必须全部通过）

### 1. 色彩一致性
- [ ] **主色偏差 ≤ 5%**：对比色值工具，确保符合 `DESIGN_GUIDE.md` 规范
  - 深邃黑 (#000000)
  - 赛博紫 (#8A2BE2)
  - 终端绿 (#00FF00)
  - 商务蓝 (#007AFF)
  - 荧光绿 (#00FF00)
  - 高级灰 (#333333)
- [ ] **RGB 值正确**：使用吸管工具检查，误差 ≤ 5 个色值单位

### 2. 文字可读性
- [ ] **标题对比度 ≥ 4.5:1**：在白色和深色背景上都清晰可读
- [ ] **正文对比度 ≥ 3:1**：在小屏幕上无障碍阅读
- [ ] **无文字锯齿**：字体边缘平滑，无像素化
- [ ] **中文字体正确**：已使用思源黑体或阿里巴巴普惠体
- [ ] **英文字体正确**：已使用 Inter 或 SF Pro
- [ ] **代码字体等宽**：已使用 JetBrains Mono 或 Fira Code

### 3. 尺寸规格
- [ ] **像素尺寸完全正确**：对比 `TASK_CHECKLIST.md` 中的规格表
- [ ] **宽高比正确**：无变形（aspect ratio 正确）
- [ ] **安全区域内**：重要元素不被裁切（预留 10% 边距）
- [ ] **响应式适配**：在不同屏幕尺寸下表现良好

### 4. 文件属性
- [ ] **格式正确**：PNG-24（透明）/ JPG（高质）/ SVG（矢量）
- [ ] **色彩模式**：RGB（屏幕）/ CMYK（印刷，如需要）
- [ ] **DPI 正确**：72（屏幕）/ 300（印刷）
- [ ] **文件大小合规**：
  - 头像 ≤ 500KB
  - 配图 ≤ 2MB
  - 封面 ≤ 5MB
  - Logo ≤ 1MB

### 5. 透明度处理
- [ ] **背景干净**：PNG 文件透明背景无残留
- [ ] **边缘平滑**：无锯齿、无毛边
- [ ] **抗锯齿开启**：导出时已启用
- [ ] **无鬼影**：无半透明残影

### 6. 命名规范
- [ ] **全小写**：文件名无大写字母
- [ ] **下划线分隔**：使用 `_` 而非空格或连字符
- [ ] **版本号正确**：`v1.0` / `v1.1` 格式
- [ ] **无特殊字符**：无 `!@#$%^&*()` 等字符
- [ ] **英文命名**：避免中文字符

### 7. 矢量可缩放
- [ ] **有矢量源文件**：Logo/图标提供 SVG/AI 源文件
- [ ] **无栅格化**：未使用位图素材
- [ ] **路径闭合**：所有矢量路径闭合，无断点
- [ ] **组内元素整齐**：图层/组结构清晰

### 8. 版权合规
- [ ] **字体有授权**：已确认字体使用许可
- [ ] **素材无版权问题**：AI 生成或购买授权
- [ ] **无水印**：文件无任何水印/签名
- [ ] **许可证文件存在**：自定义素材提供 `assets/fonts/licenses/` 文档

## 🎨 细节检查项（Detail Checks - 按需通过）

### 9. 视觉美感
- [ ] **构图平衡**：视觉重心居中，无偏斜
- [ ] **留白合理**：不过度拥挤，留白 ≥ 15%
- [ ] **对齐规范**：元素对齐网格（8px 或 16px 栅格）
- [ ] **一致性**：同一系列资产风格统一

### 10. 品牌一致性
- [ ] **Logo 正确**：使用的 Logo 版本正确（矢量/单色/彩色）
- [ ] **配色系统**：使用的颜色符合品牌规范
- [ ] **字体层级**：标题/正文字号比例合理（1.5x - 2.0x）
- [ ] **图标库**：使用的图标来自统一库（如 FontAwesome/Lucide）

### 11. 可访问性
- [ ] **色盲友好**：不只依赖颜色区分信息（加形状/图标辅助）
- [ ] **高对比度模式**：在高对比度模式下依然清晰
- [ ] **键盘导航**：如用于网页，支持键盘焦点

### 12. 性能优化
- [ ] **已压缩**：使用 TinyPNG/Compressor.io 压缩
- [ ] **渐进式加载**：JPG 文件已启用渐进式加载
- [ ] **WebP 备份**：提供 WebP 格式（用于网页）

## 🚨 常见问题及解决方案

### ❌ 问题 1：色彩偏差
**症状**：输出的紫色偏蓝或偏红
**解决**：
1. 使用吸管工具检查 RGB 值
2. 在 Photoshop 中通过 `图像 > 调整 > 色相/饱和度` 微调
3. 重新导出，关闭色彩管理（嵌入 sRGB 配置）

### ❌ 问题 2：文字锯齿
**症状**：字体边缘有锯齿或模糊
**解决**：
1. 确保使用抗锯齿（Anti-aliasing）
2. 将文字转为矢量（AI/SVG）
3. 导出时选择"高质量"或"最高质量"

### ❌ 问题 3：透明背景残留
**症状**：PNG 文件有灰色或白色残留
**解决**：
1. 在 Photoshop 中使用魔棒工具（容差 10）选中背景
2. 删除选中区域
3. 导出时选择"透明背景"

### ❌ 问题 4：文件过大
**症状**：图片文件超过规定大小
**解决**：
1. 使用 TinyPNG 在线压缩（无损）
2. 在 Photoshop 中调整导出质量（80%-90%）
3. 考虑使用 WebP 格式（体积减少 30%）

### ❌ 问题 5：命名错误
**症状**：文件名有空格或大写
**解决**：
1. 使用批量重命名工具（如 Bulk Rename Utility）
2. 遵循 `小写_下划线_v版本号.png` 格式
3. 避免中文和特殊字符
