# moneybao-agent 模仿学习指南

## 📚 这个学习系统是什么

这是为 moneybao-agent（有钱）设计的**技能学习与模仿系统**。

### 核心能力
- 📖 **知识采集**：从文章、视频号学习
- 💾 **记忆存储**：持久化存储技能卡片
- 🔍 **语义检索**：按需调用相关知识
- 🎭 **风格模仿**：应用学习到的风格和话术

---

## 🚀 快速开始

### 步骤 1：提供学习材料

#### 方式 A：文章链接
```
请帮我学习这篇文章：[链接]
- 提取风格特点
- 提取话术模式
- 生成技能卡片
```

#### 方式 B：视频号内容
```
请帮我学习这个视频号：[视频号名称/链接]
- 分析视频风格
- 提取常用话术
- 生成技能卡片
```

### 步骤 2：moneybao-agent 自动学习

moneybao-agent 会：
1. 阅读/获取内容
2. 分析风格和话术
3. 生成**技能卡片**（Skill Card）
4. 存储到**长期记忆**

### 步骤 3：应用模仿

```
请用[技能名称]的风格写一篇关于[主题]的文案
```

## 🔧 技术说明

### 记忆系统
- 使用 **memory-lancedb** 插件
- 支持向量搜索（语义检索）
- 自动记忆维护

### 检索能力
```
# 语义搜索
memory_search(query="小红书种草风格")

# 直接读取
memory_get(path="learning/001-小红书种草文案.md")
```

## ⚡ 性能优化

### 当前配置
```
✅ memory-lancedb: enabled
✅ 长期记忆: 已激活
```

### 如果检索慢
1. 检查技能卡片数量（建议 < 1000）
2. 使用更精确的搜索关键词
3. 考虑分类存储（skills/ 目录）
