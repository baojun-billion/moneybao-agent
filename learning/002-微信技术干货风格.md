## 📝 执行步骤

### 步骤 1：准备数据
```
1. 收集每个推荐项的：
   - 下载量/评分
   - 功能核心点
   - 实测效果

2. 准备截图/示例
   - 每个推荐至少 1 张图
   - 标注关键功能
```

### 步骤 2：撰写推荐
```
每个推荐项结构：
1. 小标题：[技能名称] — [一句话卖点]
2. 功能描述：2-3 句，讲清楚能干什么
3. 数据支撑：[下载量, 评分, 星标]
4. 实测结果：1 句，具体效果
5. 安装命令：clawhub install [skill-name]
```

### 步骤 3：添加价值总结
```
在文章结尾总结：
- 这几个推荐解决了什么问题
- 相比竞品的优势
- 后续会推荐更多
```

```
实测：告诉它一次偏好，以后每次对话都生效
安装命令：clawhub install self-improving-agent

[... 其他 6 个推荐 ...]

Skill 不是装得越多越好。这 8 个就够了——搜索 + 记忆 + 总结 + 天气 + 笔记 + 提醒 + 画图 + 语音转文字，已经是一个能干活的私人助手。

后面我们还会继续推荐更多好用的 Skill——效率工具、开发者专属、安全避坑指南。关注我们，不错过。
```
