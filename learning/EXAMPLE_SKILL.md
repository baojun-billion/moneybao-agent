### 语气
- [x] 热情

### 语言特点
```
- 大量使用网络用语：yyds, 绝绝子, 宝子们, 家人们
- 频繁使用表情符号：✨, 💖, 🌸, 💕, 🌟
- 短句为主，节奏快，1-2 句换行
- 反问句多：是不是很心动？谁懂？
- 感叹句多：太好看了！太喜欢了！
- 使用"体验感"词汇：亲测, 踩雷, 真的绝绝子
```

### 结构特征
```
1. 标题（15-25 字）：抓住注意力，包含关键词
2. 开场（1-2 句）：表达激动/惊喜
3. 中段（3-5 句）：讲体验/感受/细节
4. 结尾（1-2 句）：行动号召 + 标签
```

## 📝 执行步骤

### 步骤 1：准备标题
```
写 3-5 个标题，包含：
- 产品关键词
- 情感词（绝绝子, 真的会谢）
- 场景词（通勤, 约会, 撸娇）
```

### 步骤 2：开场表达
```
用 1-2 句表达第一感受：
- 激动（天啊！救命！）
- 惊喜（宝藏！绝绝子！）
- 亲身体验（亲测了！真的绝！）
```

### 步骤 3：体验分享
```
用 3-5 句分享细节：
- 使用场景（通勤、约会、日常）
- 产品感受（质感、舒适、显白）
- 对比/优势（比 xx 好用，性价比高）
```

### 步骤 4：结尾 + 标签
```
1 句行动号召 + 3-5 个标签：
- 赶冲！
- #小红书种草 #宝藏 #必入
```
