# 🎬 video-use - 视频生成与应用工作流

> 从创意到发布的一站式视频创作流水线

---

## 📖 快速开始

### 基础使用

```bash
# 用户触发
"帮我做个视频"
"生成一个产品视频"
"video-use"
```

### 工作流程

```
创意构思 → 视频生成 → 质量审核 → 内容优化 → 渠道发布 → 效果追踪
```

---

## 🎯 核心功能

### 1. 视频生成
使用 `video_generate` 工具生成高质量视频

**支持场景**：
- 📱 社交媒体内容（抖音/TikTok/视频号/小红书）
- 🎯 营销活动视频
- 📚 品牌叙事视频
- 🎓 教育培训视频

### 2. 质量审核
自主评估视频质量，识别问题和改进点

**审核标准**：
- **S 级**：可直接发布
- **A 级**：小幅修改
- **B 级**：需重新生成
- **C 级**：方向调整

### 3. 优化迭代
根据反馈调整 Prompt，重新生成或润色视频

### 4. 发布支持
提供平台适配、文案建议、发布时间规划

## 💡 Prompt 技巧

### 基础模板

```
[场景描述]，[动作/动态]，[氛围/情绪]，[风格参考]，[技术细节]
```

### 优质示例

**产品展示**：
```
一个精致的手机产品展示视频，手机在空中缓慢旋转，展示全面屏设计和玻璃背板的质感。
光线从侧面扫过，展现金属边框的光泽。背景是极简的渐变色，色调温暖高级。
4K 高清画质，慢动作镜头。
```

**品牌故事**：
```
温暖感人的品牌故事视频。从清晨的第一缕阳光开始，创业者在工作台前专注工作的场景。
镜头切换到团队协作的画面，每个人脸上洋溢着笑容。最后是产品出现在用户手中的特写，
用户满意的表情。整体色调温暖柔和，配乐温馨动人。
```

### 优化清单

- ✅ 具体场景（不要模糊描述）
- ✅ 动作描述（旋转、浮动、滑动）
- ✅ 光影细节（光线方向、强度、质感）
- ✅ 氛围营造（温暖、科技感、活力）
- ✅ 技术规格（4K、慢动作、景深）

## 🚀 使用示例

### 示例 1：快速产品展示

```
用户：帮我做一个 30 秒的产品展示视频，用于抖音。产品是智能手表。

工作流：
1. 确认需求：抖音（竖屏）、30s、智能手表、科技感
2. 生成视频：
   Prompt: "一款时尚的智能手表在空中缓慢旋转，展示表盘设计和表带质感。
   手表放在手臂上，展示佩戴效果。屏幕上显示心率、步数等健康数据。
   背景是科技感的动态元素，光线从侧面照射，突出金属质感。4K 高清，
   节奏明快，适合抖音平台。"
3. 质量审核：检查清晰度、流畅度、风格
4. 优化与交付：调整、提供文案建议
```

### 示例 2：品牌故事视频

```
用户：我们需要一个 60 秒的品牌故事视频，用于官网和视频号。

工作流：
1. 创意构思：环保科技公司、科技助力绿色生活、温暖正能量
2. 分镜头脚本（简要）：
   - 0-10s：环境问题展示
   - 10-30s：科技研发
   - 30-50s：产品应用、环境改善
   - 50-60s：品牌 Logo 和口号
3. 分片段生成视频后剪辑
4. 审核优化：叙事流畅度、情感、节奏
```

## ❓ 常见问题

### Q: 生成的视频有 AI 瑕疵怎么办？
**A**:
1. 修改 Prompt，增加更具体的描述
2. 使用参考图引导风格
3. 尝试不同的 aspectRatio 或 duration
4. 如问题严重，考虑重新构思创意

### Q: 如何提高视频一致性？
**A**:
1. 使用参考图（image/images 参数）
2. 在 Prompt 中强调风格关键词
3. 为系列视频建立固定的 Prompt 模板

### Q: 视频生成失败或质量很差？
**A**:
1. 检查 Prompt 是否过于复杂
2. 尝试缩短视频时长
3. 检查网络连接和 API 状态
4. 联系 jovibao_agent 进行技术排查

## 📊 质量标准

### 视频质量分级

**S 级（可直接发布）**：
- 视觉质量优秀，无明显 AI 瑕疵
- 动作流畅自然
- 色调风格统一
- 核心信息清晰传达

**A 级（小幅修改）**：
- 整体质量良好
- 有 1-2 个小瑕疵
- 调色或剪辑可优化

**B 级（需重新生成）**：
- AI 瑕疵较多
- 动作不自然
- 风格不统一

**C 级（方向调整）**：
- 核心创意偏离预期
- 需要重新构思方向
