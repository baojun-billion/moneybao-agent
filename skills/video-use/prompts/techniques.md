# Prompt 工程技巧集锦

## 🔑 核心原则

### 具体化原则
```
❌ 错误："做一个手机视频"
✅ 正确："一个黑色的 iPhone 15 Pro Max 在木桌上缓慢旋转，
        展示钛金属边框的光泽和瓷质背板的质感。光线从左侧 45 度角照射，
        勾勒出手机的轮廓。"

技巧：
- 描述具体产品（品牌、型号、颜色）
- 描述具体场景（材质、位置、环境）
- 描述具体动作（旋转、浮动、滑动）
- 描述具体光线（方向、强度、质感）
```

### 分层描述原则
```
从整体到细节，分层描述：

1. 整体场景
   - 在什么场景中
   - 背景是什么

2. 主体描述
   - 主体是什么
   - 核心特征

3. 动态动作
   - 如何运动
   - 运动轨迹

4. 光影细节
   - 光线方向
   - 光线强度

5. 氛围营造
   - 情绪氛围
   - 色调风格

6. 技术规格
   - 画质标准
   - 特殊效果
```

### 目标导向原则
```
明确视频的目标，围绕目标编写 Prompt：

目标：展示产品质感
→ 重点描述材质、光线、细节

目标：展示产品功能
→ 重点描述使用场景、效果

目标：营造情感共鸣
→ 重点描述人物表情、互动、氛围

目标：突出科技感
→ 重点描述科技元素、光线、色调
```

## 📝 结构模板

### 基础模板
```
[主体] + [场景] + [动作] + [光线] + [氛围] + [技术规格]

示例：
"一款黑色的智能手表（主体）
在浅色木桌上（场景）
缓慢旋转展示表盘细节（动作），
光线从上方柔和照射（光线），
营造专业高端的氛围（氛围），
4K 高清画质（技术规格）"
```

### 产品展示模板
```
"[产品名称]展示视频。
产品在[场景]中[动作描述]，
展示[核心卖点1]和[核心卖点2]。
光线从[方向]照射，突出[材质/质感]。
背景是[背景描述]，色调[风格描述]。
[技术规格]，[特殊效果]。"

示例：
"Apple iPhone 15 Pro 展示视频。
手机在空中缓慢旋转，
展示钛金属边框的光泽和瓷质背板的质感。
光线从左侧照射，勾勒出手机的轮廓。
背景是极简的深色渐变，色调科技感十足。
4K 高清画质，慢动作镜头。"
```

### 品牌故事模板
```
"[品牌故事标题]。
从[起始场景]开始，镜头切换到[场景2]，
展示[品牌理念/核心价值观]。
特写[人物/产品]的[表情/细节]，
营造[情感氛围]。
色调[风格描述]，[技术规格]。
横屏 16:9，[时长]秒。"

示例：
"温暖感人的品牌故事。
从清晨的第一缕阳光开始，
镜头切换到创业者在工作台前专注工作的场景。
特写创业者专注的表情和坚定的眼神，
营造温暖感人的氛围。
色调温暖柔和，4K 高清画质。
横屏 16:9，60 秒。"
```

### 教育培训模板
```
"[知识点名称]讲解视频。
从[引入问题]开始，逐步讲解[核心概念]。
用[动画/图解/案例]辅助理解。
特写[关键步骤/细节]。
背景简洁，色调[风格描述]。
[技术规格]，横屏 16:9，[时长]。"

示例：
"重力概念讲解视频。
从苹果落地开始，逐步讲解重力原理。
用动画演示重力的作用。
特写关键公式和实验演示。
背景简洁，色调专业可信。
4K 高清画质，横屏 16:9，3 分钟。"
```

## 🎨 描述技巧

### 光线描述
```
柔和光线：
"光线柔和地从侧前方照射，营造温暖舒适的氛围"

强烈光线：
"光线充足明亮，从正上方照射，突出细节"

侧光：
"光线从侧面 45 度角照射，勾勒出物体的轮廓"

背光：
"光线从背后照射，形成剪影效果，营造神秘感"

自然光：
"光线自然明亮，来自窗外，营造真实感"

冷光：
"光线冷峻明亮，营造科技感和未来感"
```

### 动作描述
```
旋转：
"在空中缓慢旋转 360 度，展示全貌"

浮动：
"在空中轻轻浮动，营造梦幻感"

滑动：
"在桌面上平滑滑动，展示流畅性"

展开：
"从折叠状态缓缓展开，展示过程"

升起：
"从底部缓缓升起，突出产品"

特写：
"镜头缓慢推进，特写细节"
```

### 质感描述
```
金属质感：
"金属边框光泽闪闪，质感高端"

玻璃质感：
"玻璃背板通透，光线折射出彩虹光泽"

木质感：
"木质表面温暖自然，纹理清晰"

皮革质感：
"皮革质感柔软细腻，触感真实"

陶瓷质感：
"瓷质背板光滑细腻，触感温润"
```

## 🔧 参数优化

### 视频时长
```
15-30 秒：
- 社交媒体快闪
- 产品亮点展示
- 品牌印象广告

30-60 秒：
- 产品完整展示
- 品牌故事短片
- 用户见证

60-120 秒：
- 深度品牌故事
- 教育培训片段
- 完整演示

3-15 分钟：
- 深度科普
- 完整教程
- 纪录片
```

### 比例设置
```
9:16 竖屏：
- 抖音、TikTok
- 小红书
- Reels、Shorts

16:9 横屏：
- B站
- YouTube
- 视频号（横屏内容）

1:1 方形：
- Instagram
- 社交媒体卡片
```

### 特殊效果
```
慢动作：
"慢动作镜头，展现细节美感"

浅景深：
"浅景深效果，背景虚化，突出主体"

快速剪辑：
"快速剪辑风格，每个镜头 2-3 秒"

平滑运镜：
"平滑的镜头运动，流畅自然"
```

## 💡 高级技巧

### 使用参考图
```
技巧：
1. 选择与目标风格一致的参考图
2. 在 Prompt 中强调参考图的关键元素
3. 使用多张参考图融合风格

示例：
"参考图片中的色调和构图，
展示产品在类似场景中的效果"
```

### 分场景生成
```
技巧：
1. 将长视频分解为多个场景
2. 为每个场景编写独立的 Prompt
3. 分别生成后剪辑组合

适用：
- 品牌故事视频
- 教程视频
- 纪录片
```

### 批量测试
```
技巧：
1. 用核心 Prompt 生成多个版本
2. 略微调整细节，对比效果
3. 择优使用或融合优点

示例版本：
版本1：强调光线和质感
版本2：强调动作和动态
版本3：强调氛围和情感
```

### 风格一致性
```
技巧：
1. 为系列视频建立固定模板
2. 保持色调、风格、光线一致
3. 使用相同的参考图

目的：
- 强化品牌识别
- 提升专业度
- 降低生成成本
```

## ❌ 常见错误

### 过于模糊
```
❌ 错误："做一个手机视频"
✅ 正确："一个黑色的 iPhone 在空中缓慢旋转，
        展示金属边框的光泽"
```

### 信息过载
```
❌ 错误："手机在空中缓慢旋转展示金属边框的光泽
        和瓷质背板的质感背景是深色渐变光线从左侧照射
        勾勒出轮廓色调科技感十足4K高清画质慢动作镜头
        适合抖音平台竖屏9:16..."

✅ 正确：分点描述，层次清晰
```

### 忽略光线
```
❌ 错误：没有描述光线
✅ 正确："光线从左侧 45 度角照射，
        勾勒出手机的轮廓"
```

### 风格不统一
```
❌ 错误：同一系列视频风格差异大
✅ 正确：使用固定模板和参数
```
