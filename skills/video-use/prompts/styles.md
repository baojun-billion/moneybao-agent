# 视频风格参考 Prompt 素材

## 🎨 色调风格

### 温暖柔和
```
色调温暖柔和，光线从侧前方照射，营造温馨舒适的氛围。色彩以暖色系为主，
包括米色、暖灰色、柔和的橙色和粉色。背景简洁干净，突出主体。
整体感觉亲切、温暖、治愈。
```

### 科技感
```
色调科技感十足，光线冷峻明亮，强调科技氛围。色彩以冷色系为主，
包括蓝色、紫色、青色、银灰色。背景带有科技元素，如光束、网格、数据流动。
整体感觉专业、前沿、未来感。
```

### 高级质感
```
色调高级质感，光线充足均匀，突出材质和细节。色彩以中性色为主，
包括黑色、白色、深灰色、金属色。背景极简，突出产品本身的质感。
整体感觉高端、专业、精致。
```

### 活力时尚
```
色调活力时尚，光线明亮多彩，营造动感氛围。色彩鲜艳饱满，
包括亮黄、亮橙、亮粉、亮蓝等。背景动态，充满时尚元素。
整体感觉年轻、活力、潮流。
```

### 复古怀旧
```
色调复古怀旧，光线柔和偏黄，营造年代感。色彩以低饱和度为主，
包括棕色、深绿色、酒红色、米黄色。背景带有复古元素。
整体感觉怀旧、有故事感、文艺。
```

### 清新自然
```
色调清新自然，光线自然明亮，营造舒适氛围。色彩以浅色系为主，
包括白色、浅蓝、浅绿、米色。背景简洁清新，突出自然感。
整体感觉清新、舒适、自然。
```

## 🎬 视觉风格

### 电影质感
```
电影质感的视觉风格。镜头语言专业，包括浅景深、慢动作、平滑的运镜。
色调经过精心调色，对比度适中，细节丰富。画面有电影般的质感。
整体感觉专业、艺术、高级。
```

### 动画风格
```
动画风格的视觉风格。画面流畅，色彩鲜艳饱和，人物和场景有一定的艺术夸张。
动作流畅自然，节奏明快。整体感觉活泼、有趣、富有想象力。
```

### 实拍感
```
真实感十足的实拍风格。画面自然真实，不过度修饰。光线自然，
色调偏向真实感。动作和场景贴近现实。整体感觉真实、可信、接地气。
```

### 极简主义
```
极简主义的视觉风格。画面简洁干净，元素少而精。背景单一或空白，
突出主体。色调统一，色彩克制。整体感觉简洁、专业、高级。
```

### 艺术感
```
富有艺术感的视觉风格。画面构图讲究，色彩搭配有艺术性。
可能有特殊的镜头处理或视觉效果。整体感觉艺术、有品位、独特。
```

## 🎭 情感风格

### 温暖感人
```
温暖感人的情感风格。镜头语言柔和细腻，特写情感细节。
色调温暖柔和，光线柔和。整体感觉温暖、感人、有情感共鸣。
```

### 激励人心
```
激励人心的情感风格。镜头语言有力向上，展现突破和成长。
色调明亮有力，光线充足。整体感觉激励、鼓舞、充满希望。
```

### 轻松愉快
```
轻松愉快的情感风格。镜头语言轻松活泼，节奏明快。
色调明亮鲜艳，光线充足。整体感觉轻松、愉快、有趣。
```

### 专业可信
```
专业可信的情感风格。镜头语言稳健专业，节奏适中。
色调专业稳重，光线充足。整体感觉专业、可信、权威。
```

### 神秘悬疑
```
神秘悬疑的情感风格。镜头语言低调克制，光线偏暗。
色调低饱和度，营造神秘感。整体感觉神秘、悬疑、引人入胜。
```

## 🌟 特殊效果

### 慢动作
```
慢动作镜头。动作以慢速播放，展现细节美感。配合浅景深，
突出主体。整体感觉优雅、有艺术感。
```

### 快速剪辑
```
快速剪辑风格。镜头切换快速，每个镜头 1-2 秒。配合音乐节奏，
营造动感。整体感觉节奏明快、活力十足。
```

### 浅景深
```
浅景深镜头。背景虚化，突出主体。细节清晰，质感强烈。
整体感觉专业、高端、有质感。
```

### 动态背景
```
动态背景效果。背景有动态元素，如光束、粒子、图形动画。
营造科技感或活力感。整体感觉动态、有趣、前卫。
```

### 平滑运镜
```
平滑的镜头运动。镜头移动流畅自然，没有突兀的切换。
营造沉浸感。整体感觉流畅、舒适、专业。
```

## 📱 平台适配风格

### 抖音/TikTok 风格
```
适合抖音/TikTok 风格。节奏快，前 3 秒抓眼球。
色彩鲜艳饱和，光线明亮。竖屏 9:16，15-60 秒。
整体感觉快节奏、活力、潮流。
```

### 小红书风格
```
适合小红书风格。强调美感和质感，色调清新高级。
光线柔和自然，构图讲究。竖屏 9:16，20-45 秒。
整体感觉精致、美观、有品位。
```

### B站风格
```
适合 B 站风格。内容深度高，信息密度大。
色调专业可信，制作精良。横屏 16:9，3-15 分钟。
整体感觉专业、有深度、有知识含量。
```

### 视频号风格
```
适合视频号风格。真实感强，接地气。
色调自然不过度修饰，光线自然。可横可竖，30-120 秒。
整体感觉真实、可信、接地气。
```

## 💡 风格组合技巧

### 1. 产品展示组合
```
高级质感 + 浅景深 + 电影质感
突出产品细节和品质感
```

### 2. 品牌故事组合
```
温暖感人 + 平滑运镜 + 电影质感
营造情感共鸣和高级感
```

### 3. 社交媒体组合
```
活力时尚 + 快速剪辑 + 动态背景
符合快节奏、年轻化的平台特性
```

### 4. 教育培训组合
```
专业可信 + 极简主义 + 清晰稳定
突出内容和专业性
```

### 5. 科技产品组合
```
科技感 + 浅景深 + 电影质感
展现科技感和高级质感
```
