## 💡 场景使用技巧

### 1. 真实感
```
使用真实的场景描述，让观众有代入感。
具体地点、具体环境、具体物品。
```

### 2. 情感连接
```
场景要能传递情感，让观众产生共鸣。
温暖、欢乐、专业、励志等。
```

### 3. 产品契合
```
场景要符合产品特性和使用场景。
产品的最佳使用环境是什么？
```

### 4. 品牌一致
```
场景要符合品牌调性和风格。
科技品牌用科技场景，生活品牌用生活场景。
```

### 5. 视觉美感
```
场景本身要有美感，包括光线、色调、构图。
符合观众的审美期待。
```
