## 详细分镜头脚本

### 【镜头1】教室特写
- **时长**：15秒
- **场景**：教室课桌
- **画面描述**：温暖的阳光从窗户射入，洒在课桌上。一张崭新的A4纸静静地躺在桌面上，洁白无瑕，边缘整齐。背景是模糊的教室环境，可以看到其他学生的背影。
- **镜头运动**：缓慢推近，从全景推向纸张特写
- **音效**：
  - 00:00-00:05：上课铃声（清脆）
  - 00:05-00:10：轻微的翻书声、纸张摩擦声
  - 00:10-00:15：温暖的背景音乐渐入（钢琴单音）
- **字幕/旁白**："我出生的那天，阳光很暖"
- **AI工具使用**：
  - 智谱GLM-4：生成画面描述和氛围文字
  - 待定工具：文生图生成"阳光教室"场景
- **人机协作记录**：
  ```
  人类创意：设计温暖的开场，用阳光象征希望
  AI辅助：使用智谱GLM-4优化画面描述
  提示词：温暖的阳光从窗户射入教室，洒在课桌上，一张崭新的A4纸，洁白无瑕，温馨的氛围，柔光效果
  ```

### 【镜头2】学生书写
- **时长**：20秒
- **场景**：教室课桌
- **画面描述**：学生的手握着笔，在纸上认真书写。镜头聚焦在笔尖和纸张的接触点。只看到"课堂笔记"四个字被写出来，然后笔停住了。学生的手犹豫了一下，然后把纸推到一边。
- **镜头运动**：中景，手部特写
- **音效**：
  - 笔尖划过纸张的沙沙声（清晰）
  - 学生轻微的叹息声
  - 背景音乐继续，略微变低沉
- **字幕/旁白**："他只用了我的0.1%，就把我抛弃了"
- **AI工具使用**：
  - 智谱GLM-4：优化旁白文案，使其更具情感
  - 待定工具：动画生成写字过程
- **人机协作记录**：
  ```
  人类创意：用"0.1%"这个数据体现浪费
  AI辅助：智谱GLM-4优化旁白："他只用了我的0.1%，就把我抛弃了"
  ```

### 【镜头3】被揉成团
- **时长**：10秒
- **场景**：课桌到垃圾桶
- **画面描述**：纸张被学生慢慢揉成一团。镜头跟踪纸团的运动轨迹，看着它从课桌上滚落，掉进垃圾桶。纸团在垃圾桶里继续滚动，最终停在角落。
- **镜头运动**：跟随镜头，跟踪纸团运动
- **音效**：
  - 揉纸的"沙沙"声
  - 垃圾桶盖打开和关闭的"哐当"声
  - 悲伤的钢琴声渐强（半音阶下行）
- **字幕/旁白**："在黑暗的角落，我遇见了它们"
- **AI工具使用**：
  - 智谱GLM-4：生成揉纸过程的动态描述
  - 待定工具：AI动画生成纸张变形过程
- **人机协作记录**：
  ```
  人类创意：用"黑暗的角落"营造孤独感
  AI辅助：智谱GLM-4生成纸团运动的细节描述
  ```

### 【镜头4】垃圾桶内的会议
- **时长**：30秒
- **场景**：垃圾桶内部
- **画面描述**：垃圾桶内，多个纸团聚在一起，每个纸团都有不同的表情和姿态。纸团A皱巴巴的，低着头；纸团B还算平整，有些不甘心；纸团C最破旧，带着沧桑感。它们围成一圈，像是在开会。
- **镜头运动**：环绕镜头，逐个展示每个纸团的表情
- **音效**：
  - 低沉的对话声（拟人化配音）
  - 纸团A（声音沮丧）："我才写了两个字就被扔了..."
  - 纸团B（声音不甘）："我是试卷背面，本来可以写草稿"
  - 纸团C（声音沧桑）："我见过很多同伴，大多数都没被珍惜"
  - 背景音乐：悲伤的钢琴旋律
- **字幕/旁白**：（无，用对白代替）
- **AI工具使用**：
  - 智谱GLM-4：优化三个纸团的对白，体现不同性格
  - 待定工具：AI图像生成纸团拟人化表情
  - 待定工具：AI语音生成三个不同角色的配音
- **人机协作记录**：
  ```
  人类创意：设计三个纸团不同的性格和经历
  AI辅助：智谱GLM-4优化对白，使其更自然、更有个性
  提示词生成：为每个纸团设计独特的表情和姿态
  ```

### 【镜头5】回忆闪回
- **时长**：45秒
- **场景**：多个场景快速切换
- **画面描述**：快节奏闪回各个纸团的前世今生。
  - 纸团A：被用来画了一幅画，只画了几笔就揉掉（5秒）
  - 纸团B：试卷背面，本来可以写草稿，但被撕下后直接丢弃（10秒）
  - 纸团C：作业本空白页被撕下，还没写一个字就扔了（10秒）
  - 其他纸团：打印错误一页、复印多余一页、宣传单看一眼就扔（20秒）
- **镜头运动**：快速切换，每个场景2-5秒
- **音效**：
  - 快节奏的打击乐，配合闪回节奏
  - 每个场景的音效：画笔声、撕纸声、打印机声、纸张落地声
  - 背景音乐：紧张的节奏，营造压抑感
- **字幕/旁白**："我们都有梦想，却没能被完成"
- **AI工具使用**：
  - 智谱GLM-4：生成多个浪费场景的描述
  - 待定工具：AI生成多个场景的画面
  - 待定工具：AI智能剪辑，制作快闪效果
- **人机协作记录**：
  ```
  人类创意：用快闪展现普遍的浪费现象
  AI辅助：智谱GLM-4生成6个不同浪费场景的详细描述
  AI剪辑：使用智能剪辑工具制作快闪转场
  ```

### 【镜头6】希望之光
- **时长**：30秒
- **场景**：垃圾桶到回收车
- **画面描述**：垃圾回收车来了，清洁工打开垃圾桶，将纸团们倒入回收箱。就在纸团们即将进入回收箱的瞬间，一束阳光照进来，照亮了每个纸团。纸团们的表情从悲伤转为希望。
- **镜头运动**：仰拍镜头，从纸团视角看阳光
- **音效**：
  - 回收车的机械声（缓慢有力）
  - 阳光照入的"闪亮"音效
  - 希望的旋律渐强（从低沉到明亮）
- **字幕/旁白**："但今天，我们要重生了！"
- **AI工具使用**：
  - 智谱GLM-4：生成希望之光的画面描述
  - 待定工具：AI生成阳光照射效果
  - 待定工具：真实拍摄回收车，AI合成阳光效果
- **人机协作记录**：
  ```
  人类创意：用阳光象征希望和重生
  AI辅助：智谱GLM-4生成阳光照入垃圾桶的详细画面描述
  视觉效果：AI添加光效，从暗到明的色彩变化
  ```

### 【镜头7】回收厂之旅
- **时长**：30秒
- **场景**：造纸回收厂
- **画面描述**：纸团进入回收厂，经历完整的造纸流程：
  - 水力碎浆（8秒）：纸团在水中被分解成纤维
  - 清洗过滤（8秒）：纤维被清洗，去除杂质
  - 烘干压制（7秒）：湿纸浆被烘干，压成纸
  - 卷纸成型（7秒）：新纸卷成筒
  整个过程充满科技感，使用冷色调（蓝、绿）
- **镜头运动**：平移镜头，跟随纸张流动
- **音效**：
  - 水流声（连续）
  - 机器运转的节奏声
  - 科技感背景音乐（电子音效）
- **字幕/旁白**："水冲去了我的旧颜，火赋予我新生"
- **AI工具使用**：
  - 智谱GLM-4：生成造纸流程的详细描述
  - 待定工具：AI动画生成造纸流程（科技感）
  - 或：查找真实造纸厂视频素材，AI标注和加速
- **人机协作记录**：
  ```
  人类创意：展示科学、现代的造纸工艺
  AI辅助：智谱GLM-4生成造纸流程的4个步骤描述
  视觉风格：AI生成科技感的蓝色调画面
  ```
